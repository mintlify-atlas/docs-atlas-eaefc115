The `/ads meta` command runs a full Meta Ads audit covering Facebook and Instagram campaigns, including Advantage+ Sales campaigns. It evaluates 50 checks organized across four weighted categories and produces a `META-ADS-REPORT.md` with a health score, an EMQ (Event Match Quality) improvement roadmap, creative fatigue alerts, and quick wins. The audit is built around Meta's post-iOS 14.5 reality: server-side tracking and creative diversity are the two highest-leverage levers available.

## Scope

`/ads meta` covers all standard campaign types and Advantage+ formats:

- Standard prospecting and retargeting campaigns on Facebook and Instagram
- Advantage+ Sales campaigns (formerly Advantage+ Shopping Campaigns, renamed early 2025)
- Advantage+ Audience, Advantage+ Creative enhancements, and Advantage+ Placements
- All placement surfaces including Feed, Reels, Stories, and Threads (GA January 2026)
- Special Ad Category compliance for housing, employment, credit, and financial products

    **10 checks: M01–M10, plus M-AT1 (v1.5)**

    The highest-weighted category because 87% of advertisers have poor Event Match Quality scores, and broken tracking makes every optimization decision unreliable. The Conversions API is mandatory post-iOS 14.5—client-side only means 30–40% data loss.

    | Check | Description | Severity |

    | M01 | Meta Pixel installed and firing on all pages | Critical |
    | M02 | Conversions API (CAPI) active alongside Pixel | Critical |
    | M03 | Event deduplication via event_id matching (≥90% dedup rate) | Critical |
    | M04 | Event Match Quality ≥8.5 for Purchase (see EMQ table below) | Critical |
    | M05 | Business domain verified in Business Manager | High |
    | M06 | Aggregated Event Measurement (AEM) top 8 events configured and prioritized | High |
    | M07 | Standard events used (not custom events replacing standard ones) | High |
    | M08 | CAPI Gateway deployed for simplified server-side tracking | Medium |
    | M09 | Attribution window: 7-day click / 1-day view configured | High |
    | M10 | Events firing in real-time (no >1hr lag in Events Manager) | Medium |
    | M-AT1 | Offline Conversions API not in use (discontinued May 2025; migrate to CAPI with action_source="physical_store") | High |

    **EMQ score targets:**

    | Event | Pass | Warning | Fail |
    |-------|------|---------|------|
    | Purchase | ≥8.5 | 6.0–7.9 | &lt;6.0 |
    | AddToCart | ≥6.5 | 4.0–6.4 | &lt;4.0 |
    | PageView | ≥5.5 | 3.0–5.4 | &lt;3.0 |

    Case study: improving EMQ from 8.6 to 9.3 produced CPA −18%, match rate +24%, ROAS +22%.

  <Tab title="Creative — 30%">
    **12 checks: M25–M32, M-CR1–M-CR4, plus M-AN1 (v1.5)**

    Creative is the #1 performance lever on Meta since Andromeda launched in October 2025. Andromeda filters tens of millions of ads down to roughly 1,000 candidates using models that are 10,000× more complex than the previous system. Ads with a Creative Similarity Score >60% across an ad set receive retrieval suppression—they compete with each other instead of reaching new audiences.

    | Check | Description | Severity |
    |-------|-------------|----------|
    | M25 | Creative format diversity: ≥3 formats active (static image, video, carousel) | Critical |
    | M26 | Creative volume: ≥10 creatives for Advantage+ Sales, ≥5 for standard | High |
    | M27 | 9:16 vertical video present for Reels/Stories | High |
    | M28 | No creatives with CTR drop >20% over 14 days (Andromeda compressed creative lifespan to 2–4 weeks) | Critical |
    | M29 | Video hook rate: &lt;50% skip rate in first 3 seconds | High |
    | M30 | Top organic posts boosted as partnership ads | Medium |
    | M31 | ≥30% of creative assets are UGC or social-native | High |
    | M32 | Advantage+ Creative enhancements enabled and tested | Medium |
    | M-CR1 | New creative tested within last 14–21 days | High |
    | M-CR2 | Prospecting frequency (7-day) &lt;3.0 per ad set | High |
    | M-CR3 | Retargeting frequency (7-day) &lt;8.0 per ad set | Medium |
    | M-CR4 | CTR ≥1.0% | High |
    | M-AN1 | Andromeda creative diversity: genuinely distinct concepts, messaging motivators, visual styles (Similarity Score &lt;60%) | Critical |

    <Warning>
      **The Andromeda rule:** 100 minor variations equal no better than 10 genuinely distinct creatives. Color swaps, text overlays on the same background, and slightly different product angles all cluster together under the same Entity ID. Andromeda suppresses the whole cluster. Different value propositions, hooks, visual styles, and talent represent true diversity.
    </Warning>

## Quality rules enforced

<CardGroup cols={2}>
  <Card title="Budget sufficiency" icon="dollar-sign">
    Every ad set must have a daily budget ≥5× the target CPA. Ad sets below this threshold cannot exit the learning phase, which means all other optimizations are pointless until budget is corrected.
  </Card>
  <Card title="Andromeda creative diversity" icon="paintbrush">
    Accounts with fewer than 10 genuinely distinct creatives are flagged. Distinctness is assessed by concept, messaging motivator, visual style, and hook—not asset count. Research shows different motivators unlock new audiences 89% of the time.
  </Card>
</CardGroup>

## Advantage+ assessment

When Advantage+ features are active, the audit evaluates:

- **Advantage+ Sales**: catalog connected, existing customer budget cap removed (February 2025), 22% higher ROAS and 11.7% CPA improvement on average
- **Advantage+ Audience**: performance vs. manual audience compared (A/B test required for accurate assessment)
- **Advantage+ Creative**: text, brightness, and image enhancement evaluations enabled
- **Advantage+ Placements**: all placement surfaces enabled unless specific exclusions are justified
- **Threads placement**: enabled within Advantage+ Placements and performance monitored

## Key thresholds

| Metric | Pass | Warning | Fail |
|--------|------|---------|------|
| EMQ (Purchase) | ≥8.5 | 6.0–7.9 | &lt;6.0 |
| Event dedup rate | ≥90% | 70–90% | &lt;70% |
| Creative formats active | ≥3 | 2 | 1 |
| Creatives per ad set | ≥5 | 3–4 | &lt;3 |
| Prospecting frequency (7-day) | &lt;3.0 | 3.0–5.0 | >5.0 |
| Retargeting frequency (7-day) | &lt;8.0 | 8.0–12.0 | >12.0 |
| CTR | ≥1.0% | 0.5–1.0% | &lt;0.5% |
| Budget per ad set | ≥5× CPA | 2–5× CPA | &lt;2× CPA |
| Learning Limited ad sets | &lt;30% | 30–50% | >50% |

## Quick wins

| Check | Fix | Time |
|-------|-----|------|
| M02: CAPI setup | Deploy via CAPI Gateway | 15 min |
| M05: Domain verification | Verify domain in Business Manager | 5 min |
| M09: Attribution window | Set to 7-day click / 1-day view | 2 min |
| M23: Exclusion audiences | Create purchaser Custom Audience, exclude from prospecting | 10 min |
| M25: Format diversity | Add video or carousel to single-image-only ad sets | 15 min |
| M39: UTM parameters | Add UTM template at campaign level | 5 min |
| M35: Attribution setting | Switch from 1-day click to 7-day click / 1-day view | 2 min |

## Output

Running `/ads meta` produces:

- **`META-ADS-REPORT.md`**: Full 50-check results with per-category scores and pass/warning/fail table
- **EMQ improvement roadmap**: Specific parameters to add (email, phone, name, city/state/zip, external_id) ranked by match rate impact
- **Creative fatigue alerts**: Every creative with CTR declining >20% over 14 days
- **Quick Wins**: Sorted by severity × estimated impact
- **Advantage+ adoption recommendations**: Which features to test and in what order
